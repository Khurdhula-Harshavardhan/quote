# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.3] - 2025-11-13

### Added
- Cross-platform build system with Makefile
- Platform detection for macOS and Linux
- Homebrew formula for macOS distribution
- GitHub Actions workflow for automated releases
- Pre-built binary support for Linux and macOS (Intel & ARM)
- Comprehensive build and installation documentation

### Fixed
- Binary compatibility issues between Linux and macOS
- Build process now works correctly on both platforms

### Changed
- Updated installation instructions for multiple platforms
- Improved dependency checking

## [0.0.2] - 2025-11-XX

### Changed
- Updated watch mode refresh interval default

## [0.0.1] - 2025-11-XX

### Added
- Initial release
- Real-time stock quote fetching
- Watch mode for continuous monitoring
- Support for international exchanges
- Beautiful terminal UI with formatting
